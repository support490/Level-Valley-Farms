Building a SAGE 50-level double-entry accounting system for Level Valley Farms chicken operations.

**Why:** The farm needs to track expenses by flock ID from chick placement through egg sale, with comprehensive reporting.

**How to apply:** Every feature should center around flock ID as the primary tracking unit. All financial entries use double-entry (debits = credits). UI should use dark glass theme with SearchSelect dropdowns for all entity references. Tech stack: Python/FastAPI + React/Tailwind/Vite + SQLite.
